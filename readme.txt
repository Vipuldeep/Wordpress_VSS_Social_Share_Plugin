Just another social share plugin. Simple but flexible.

== Description ==

Basic Social sharing data plugin for assignment purpose CMS.

== Installation ==

1. Upload the entire `VSS-social_Share` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add shortcode "[vss_social]" in Post/Page to share.
